# MotorSecurity
Um sistema de segurança para motoristas que utiliza de visão computacional e inteligência artificial integrada para capturar os movimentos do seu rosto e emitir quetro tipos diferentes de alertas visuais e sonoros.

Funcionalidades

- Detecção de Sonolência - Olhos fechados por tempo prolongado
- Detecção de Bocejo - Cansaço e fadiga do motorista  
- Detecção de Celular - Mão próxima à orelha
- Detecção de Distração - Cabeça virada para os lados
- Sistema de Voz - Alertas em tempo real
- Interface Moderna - PyQt5 com design profissional

  Tecnologias

- Python 3.8+
- OpenCV - Visão computacional
- MediaPipe - Detecção facial
- PyQt5 - Interface gráfica
- Arduino - Alertas físicos
